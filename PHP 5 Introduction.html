 PHP 스크립트는 서버에서 실행됩니다.

What You Should Already Know

 계속 진행하려면, 다음의 것들에 대한 기본적인 이해가 필요합니다.:

  • HTML
  • CSS
  • JavaScript

 What is PHP?

  • PHP 는 "PHP Hypertext Preprocessor" 의 약자입니다.
  • PHP 는 널리 사용되는 오픈 소스 스크립트 언어입니다.
  • PHP 스크립트는 서버에서 실행된다.
  • PHP 는 다운로드하여 무료로 사용할 수 있다.
  PHP 는 초보자에게는 간단하며, 또한 전문 프로그래머를 위한 다양한 고급 기능을 제공합니다.

What is a PHP File?

  • PHP 파일은 텍스트, HTML, CSS, JavaScript 및 PHP 코 드를 포함 할 수 있습니다
  • PHP 코드는 서버에서 실행되고, 그 결과가 일반 HTML로 브라우저에 반환된다
  • PHP 파일은 ".php" 확장자를 가진다.

What Can PHP Do?

  • PHP 는 동적 페이지의 콘텐츠를 생성 할 수 있습니다.
  • PHP 는 서버에 파일을 생성, 열기, 읽기, 쓰기, 삭제 및  닫을 수 있습니다.
  • PHP 는 form 데이터를 수집 할 수 있습니다.
  • PHP 는 cookies 를 보내고 받을 수 있습니다.
  • PHP 는 데이터베이스의 데이터를 추가, 삭제, 수정할 수 있습니다.
  • PHP 는 웹 사이트의 일부 페이지에 액세스하는 사용자를 제한 할 수 있습니다.
  • PHP 는 데이터를 암호화 할 수 있습니다.

PHP 는 출력이 HTML만으로 한정되지는 않는다. 이미지, PDF 파일, 심지어 플래시 무비를 출력 할 수 있습니다. 당신은 또한 XHTML 및 XML과 같은 텍스트를 출력 할 수도 있습니다.

Why PHP?

  • PHP 는 다양한 플랫폼에서 실행된다 (Windows, Linux, Unix, Mac OS X, etc.)
  • PHP 는 오늘날 사용되는 거의 모든 서버와 호환된다 (Apache, IIS, etc.)
  • PHP 는 넓은 범위의 데이터베이스를 지원
  • PHP 는 무료입니다. 공식 PHP 리소스에서 다운로드한다: www.php.net
  • PHP 는 배우기 쉽고, 서버 측에서 효율적으로 실행된다.

What Do I Need?

PHP 사용을 시작하려면, 다음을 수행한다 :

  • PHP 와 MySQL을 지원하는 웹 호스트를 찾거나,
  • 자신의 PC에 웹 서버를 설치 한 후, PHP 와 MySQL를 설치한다.

Use a Web Host With PHP Support

서버가 PHP에 대한 지원이 활성화 되어 있는 경우에는, 당신은 아무것도 할 필요가 없습니다.

다만, 몇몇 .php 파일을 만들고, 웹 디렉토리에 배치하면, 서버가 자동적으로 처리합니다.

당신은 아무것도 컴파일하거나, 별도의 툴을 설치할 필요가 없습니다.

PHP는 무료이므로, 대부분의 웹 호스트는 PHP 지원을 제공합니다.

Set Up PHP on Your Own PC

서버가 PHP를 지원하지 않는 경우, 다음을 수행해야 합니다 :

  • install a web server
  • install PHP
  • install a database, such as MySQL

공식 PHP 웹 사이트 (PHP.net) 에는 PHP 설치 지침이 제공 됩니다.: http://php.net/manual/en/install.php

 Tip: Windows 에서 PHP 구동해서 즉시 실행하려면, 다음을 수행 합니다:  Download WebMatrix
