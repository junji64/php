include (또는 require) 문(statement)은 지정된 파일에 있는 모든 텍스트/코드/마크업들을 가져다가 include 문을 사용한 파일에 복사합니다.

파일을 include 하는 것은 웹사이트의 여러 페이지에 같은 PHP, HTML, 또는 텍스트를 포함시키는데 매우 유용합니다.

PHP include 와 require Statements

include 또는 require 문을 사용하여, 하나의 PHP 파일의 내용을 또 다른 PHP 파일에 삽입할수 있다 (서버가 그것을 실행하기 전에).

include 와 require 문은 실패할 경우를 제외하고는 똑 같다.:

  • require 는 치명적 에러(E_COMPILE_ERROR)를 생성하고 스크립트를 중지한다.
  • include 는 경고(E_WARNING)를 생성하고 스크립트는 계속된다.

따라서, include 파일이 없는 경우에도 계속 실행이 되고 사용자에게 출력을 보여주고 싶으며 include 문을 사용합니다. 그렇지 않으면, FrameWork, CMS, 또는 복잡한 PHP 응용 코딩의 경우 실행의 흐름에 키(key) 파일을 포함시키기 위해서 항상 require 문을 사용한다. 이는 단지 하나의 키 파일이 실수로 누락된 경우에, 당신의 응용 프로그램의 보안과 무결성을 위태롭게 하지 않는데 도움을 준다. 

파일을 Including 하는 것은 작업을 많이 절약하게 해 줍니다. 이는 당신의 모든 웹페이지를 위한 표준 머리글, 바닥글, 또는 메뉴 파일을 만들수 있다는 것을 의미합니다. 그리고, 머리글이 업데이트 될 필요가 있을 때는, 단지 머리글 include 파일만을 업데이트하면 됩니다. 

Syntax

include 'filename';

or

require 'filename';

PHP include Examples

Example 1

"footer.php" 라고 부르는 표준 바닥 글 파일이 있으며, 다음과 같다고 가정하자:

<?php
echo "<p>Copyright &copy; 1999-" . date("Y") . " W3Schools.com</p>";
?>

페이지에 그 바닥 글 파일을 포함하려면, include 문을 사용한다:

Example

<html>
<body>

<h1>Welcome to my home page!</h1>
<p>Some text.</p>
<p>Some more text.</p>
<?php include 'footer.php';?>

</body>
</html>

Run example »

Example 2

"menu.php" 라고 부르는 표준 메뉴 파일이 있다고 가정하자 :

<?php
echo '<a href="/default.asp">Home</a> -
<a href="/html/default.asp">HTML Tutorial</a> -
<a href="/css/default.asp">CSS Tutorial</a> -
<a href="/js/default.asp">JavaScript Tutorial</a> -
<a href="default.asp">PHP Tutorial</a>';
?>

웹사이트의 모든 페이지는 메뉴 파일을 사용해야 합니다. 여기에 그것이 어떻게 가능한 지를 보여줍니다 (메뉴가 나중에 CSS로 쉽게 스타일될 수 있도록 <div> 요소를 사용합니다).:

Example

<html>
<body>

<div class="menu">
<?php include 'menu.php';?>
</div>

<h1>Welcome to my home page!</h1>
<p>Some text.</p>
<p>Some more text.</p>

</body>
</html>

Run example »

Example 3

몇 개의 변수가 정의된 "vars.php" 라는 파일이 있다고 가정하자 :

<?php
$color='red';
$car='BMW';
?>

그러면, "vars.php" 파일을 포함함으로써 함수 호출에 그 변수들이 사용될 수 있다.:

Example

<html>
<body>

<h1>Welcome to my home page!</h1>
<?php include 'vars.php';
echo "I have a $color $car.";
?>

</body>
</html>

PHP include vs. require

require 문 또한  PHP 코드 안에 파일을 포함시키는데 사용된다.

그러나, include 와 require 사이에는 큰 차이가 하나 있다;   include 문으로 파일 하나를 포함시키고, PHP가 그것을 찾을 수 없을 때, 스크립트는 계속 실행된다. :

Example

<html>
<body>

<h1>Welcome to my home page!</h1>
<?php include 'noFileExists.php';
echo "I have a $color $car.";
?>

</body>
</html>

Run example »

require 문을 사용하여 같은 예제를 수행 할 때, require 문이 치명적 에러를 반환하여, 스크립트 실행이 멈추기 때문에 echo 문이 실행 되지 않을 것이다. :

Example

<html>
<body>

<h1>Welcome to my home page!</h1>
<?php require 'noFileExists.php';
echo "I have a $color $car.";
?>

</body>
</html>

Run example »

 응용프로그램에 파일이 필요할 경우에는 require 를 사용하시오.

  파일이 필수는 아니며, 응용프로그램이 파일을 찾을 수 없더라도 계속 실행되야 한다면 include 를 사용하시오.
