내장되어 있는 DOM 파서(parser)는 PHP 에서 XML 문서를 처리할 수 있게 해준다.

What is DOM?

W3C DOM 은 HTML 과 XML 문서의 객체의 표준 세트와 그들을 조작하는데 필요한 표준 인터페이스를 제공합니다.

W3C DOM 다른 부분 (Core, XML, and HTML)과 다른 레벨(DOM Level 1/2/3)들로 분리된다.:

  • Core DOM - 모든 구조화된 문서를 위한 객체의 표준 세트를 정의 
  • XML DOM -  XML 문서를 위한 객체의 표준 세트를 정의
  • HTML DOM - HTML 문서를 위한 객체의 표준 세트를 정의

XML DOM 에 대한 자세한 내용을 학습하기 원한다면 이곳을 방문하시오 ===> XML DOM tutorial.

XML DOM Parser 

DOM 파서는 트리-기반 파서이다.

다음의 XML 문서의 일부분을 보시오:

<?xml version="1.0" encoding="UTF-8"?>
<from>Jani</from>

XML DOM 은 위의 XML 을 트리 구조로 봅니다.:

  • Level 1: XML 문서
  • Level 2: 루트(Root) 요소: <from>
  • Level 3: 텍스트 요소: "Jani"

Installation

DOM XML 파서 함수는 PHP 코어의 일부이다. 이 함수들을 사용하는데 어떤 설치로 필요치 않습니다.

XML File

아래의 XML 파일 "note.xml" 은 우리의 예에서 사용될 겁니다.:

<?xml version="1.0" encoding="UTF-8"?>
<note>
<to>Tove</to>
<from>Jani</from>
<heading>Reminder</heading>
<body>Don't forget me this weekend!</body>
</note>

Load and Output XML

우리는 XML 파서를 초기화하고, xml 을 로드하고, 출력합니다.:

Example

<?php
$xmlDoc = new DOMDocument();
$xmlDoc->load("note.xml");

print $xmlDoc->saveXML();
?>

위의 코드의 출력은 다음과 같을 것입이다:

Tove Jani Reminder Don't forget me this weekend!

브라우져 창에서 "View source"를 선택한다면, 다음의 HTML을 보게 될 것입니다. :

<?xml version="1.0" encoding="UTF-8"?>
<note>
<to>Tove</to>
<from>Jani</from>
<heading>Reminder</heading>
<body>Don't forget me this weekend!</body>
</note>

위의 예는 DOMDocument-Object 를 생성하고 "note.xml" 로부터 XML 을 로드합니다.

그리고 saveXML() 함수는 내부의 XML 문서를 문자열로 두고, 우리는 그것을 출력할 수 있습니다.

Looping through XML

우리는 XML 파서를 초기화하고, XML을 로드하며, <note> 요소의 모든 요소들에 대하여 반복합니다.:

Example

<?php
$xmlDoc = new DOMDocument();
$xmlDoc->preserveWhiteSpace = false;
$xmlDoc->load("note.xml");

$x = $xmlDoc->documentElement;
foreach ($x->childNodes AS $item)
  {
  print $item->nodeName . " = " . $item->nodeValue . "<br>";
  }

?>

위의 코드의 출력은  다음과 같을 것이다:

to = Tove
from = Jani
heading = Reminder
body = Don't forget me this weekend!

XML을 생성할 때, 노드들 사이에 공백이 종종 포함됩니다. XML DOM 파서는 이들을 보통의 노드로 간주하며, 이를 인식하지 못하면, 그들이 종종 문제를 발생시킵니다.
