PHP image uploaing & displaying using MySQL


다음의 예는 image 파일을 DB 에 업로딤하고 DB에 있는 것을 표시하는 방벙을 보여준다.
  • image.php : local 컴퓨터에 있는 image 파일을 서버에 업로드 하기 위한 form 요소와 완료되었을 때 get.php 를 이용하여 해당 image 를 가져와 보여줌. (More complete source code) : image.php / delete_img.php
  • get.php  : 전달 받은 image  파일 이름에 해당하는 이미지를 DB에서 추출해 전달하는 역활을 함
  • MySQL : database "my_db" 와 table "images"
                 e.g.) CREATE TABLE images(
                        id INT NOT NULL AUTO_INCREMENT,
                        PRIMARY KEY(id),
                        name varchar(100), data MEDIUMBLOB);

get.php

아래의 코드(get.php)는 image.php 로부터 image 파일 name 을 전달 받아 DB로부터 해당 이름을 갖는 image 를 추출해서 반환한다.

<?php
$con = mysqli_connect("localhost","root","123123","my_db") or die(mysqli_connect_error());
$name = $_REQUEST['name'];
$image = mysqli_query($con, "SELECT * FROM images WHERE name='$name'");
$image = mysqli_fetch_assoc($image);
$image = $image['data'];
// header("Content-type: image/jpeg");
echo $image;
?>
