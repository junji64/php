PHP에는 숫자에 대한 수학적 작업을 수행 할 수있는 수학 함수 세트가 있습니다.

PHP Pi() 함수

pi()함수는 PI 값을 반환합니다.

Example

<?php
echo(pi()); // returns 3.1415926535898
?>

Run example »

PHP min() 과 max() 함수

min()및 max() 함수는 인수 목록에서 가장 낮은 또는 높은 값을 찾을 수 있습니다 :

Example

<?php
echo(min(0, 150, 30, 20, -8, -200)); // returns -200
echo(max(0, 150, 30, 20, -8, -200)); // returns 150
?>

PHP abs() 함수

abs() 함수는 숫자의 절대 (양수) 값을 반환합니다.

Example

<?php
echo(abs(-6.7)); // returns 6.7
?>

PHP sqrt() 함수

sqrt() 함수는 숫자의 제곱근을 반환합니다.

Example

<?php

PHP round() 함수

round() 함수는 부동 소수점 숫자를 가장 가까운 정수로 반올림합니다.

Example

<?php

echo(round(0.49)); // returns 0

rand() 함수는 난수를 생성합니다.

Example

PHP Constants

상수는 단순한 값을 위한 식별자(이름)입니다. 그 값은 스크립트 안에서 변경 될 수 없다.

유효한 상수 이름은 문자나 밑줄(underscore)로 시작합니다 (상수 이름 앞에는 $ 기호가 없습니다.).

Note: 변수와는 달리 상수는 스크립트 전체에서 자동으로 전역범위를 갖는다. 

Set a PHP Constant

상수(Constants)는 한번 정의되면 그들을 "변경"하거나 "정의되지 않은"(undefined) 것으로 할 수 없는 것을 제외하고는 변수(variables)와 같다.

상수를 설정하려면, 세 가지 매개 변수를 취하는 define() 함수를 사용한다. : 첫 번째 매개 변수는 상수의 이름을 정의, 두 번째 매개 변수는 상수의 값을 정의, 세번째 매개 변수는 상수 이름을 대소문자 구별 안할 지(case-insensitive) 여부를 명시하며, 기본 값은 false이다.

아래의 예제는 "Welcome to W3Schools.com!" 의 값을 대소문자 구분하는 상수(case-sensitive constant)를 생성한다.:

Example

<?php
define("GREETING", "Welcome to W3Schools.com!");
echo GREETING;
?>

Run example »

아래의 예제는 "Welcome to W3Schools.com!" 의 값을 대소문자 구분하지 않는 상수(case-insensitive constant)를 생성한다.: 

Example

<?php
define("GREETING", "Welcome to W3Schools.com!", true);
echo greeting;
?>
