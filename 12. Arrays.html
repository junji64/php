What is an Array?

배열(array)은 한번에 하나 이상의 값들을 담을 수 있는 특별한 변수입니다.

만약 항목들의 목록(예를 들어 자동차 이름 목록)을 가지고 있다면, 하나의 변수에 차 하나를 저장하는 것은 다음과 같이 할 수 있습니다. :

$cars1="Volvo";
$cars2="BMW";
$cars3="Toyota";

그러나, 당신이 자동차들을 하나씩 살펴보면서 특정한 차 하나를 찾고자 한다면? 그리고 차가 3개가 아니라 300개 라면?

해답은 배열을 만드는 것입니다.

배열은 하나의 이름에 여러 값들을 저장할 수 있고, 인덱스 번호를  참조하여 값에 접근 할 수 있습니다.

PHP에서 배열 만들기

PHP 에서는 array() 함수를 사용하여 배열을 만든다.:

array();

PHP 에는 배열의 세 가지 유형이 있다:

  • Indexed arrays - 숫자 인덱스를 갖는 배열
  • Associative arrays - 이름 키(named keys)를 갖는 배열
  • Multidimensional arrays - 하나 이상의 배열을 포함하는 배열

PHP Indexed Arrays

인덱스 배열을 만드는 두 가지 방법이 있습니다. :

인덱스(index)는 자동적으로 할당될 수 있습니다. (인덱스는 항상 0부터 시작) :

$cars=array("Volvo","BMW","Toyota");

또는 인덱스를 수동으로 할당 할 수 도 있습니다.:

$cars[0]="Volvo";
$cars[1]="BMW";
$cars[2]="Toyota";

다음의 예는, $cars 라는 이름의 인덱스 배열을 생성하고, 세 개의 요소를 할당한 다음, 배열의 값들에 포함된 텍스트를 출력합니다. :

Example

<?php
$cars=array("Volvo","BMW","Toyota");
echo "I like " . $cars[0] . ", " . $cars[1] . " and " . $cars[2] . ".";
?>

배열의 길이 얻기 - count() 함수

count() 함수는 배열의 길이(요소의 개수)를 반환하는데 사용됩니다.:

Example

<?php
$cars=array("Volvo","BMW","Toyota");
echo count($cars);
?>

Run example »

인덱스 Array 에서의 루프 반복

인덱스 배열의 모든 값들을 출력하는데, 다음과 같이 for 루프를 사용할 수 있다.:

Example

<?php
$cars=array("Volvo","BMW","Toyota");
$arrlength=count($cars);

for($x=0;$x<$arrlength;$x++)
  {
  echo $cars[$x];
  echo "<br>";
  }
?>

Run example »

PHP 연계 배열(Associative Arrays)

연계 배열(Associative arrays)은 값을 할당하는데 이름 키(named keys)를 사용하는 배열이다.

연계 배열을 생성하는 두 가지 방법이 있습니다.: 

$age=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");

or:

$age['Peter']="35";
$age['Ben']="37";
$age['Joe']="43";

그러면, 이름 키는 다음처럼 스크립트에서 사용될 수 있다.:

Example

<?php
$age=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");
echo "Peter is " . $age['Peter'] . " years old.";
?>

Run example »

연계 배열(Associative Array)을 통한 루프 반복

연계 배열의 모든 값들을 반복하여 출력하려면,  다음과 같이 foreach 루프를 사용할 수 있다. :

Example

<?php
$age=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");

foreach($age as $x=>$x_value)
  {
  echo "Key=" . $x . ", Value=" . $x_value;
  echo "<br>";
  }
?>

PHP - Multidimensional Arrays

때때로 하나 이상의 키(key)로 값을 저장하고 싶을 수 있습니다.

이는 다차원 배열(multidimensional arrays)에 저장될 수 있다.

다차원 배열은 하나 이상의 배열을 포함하는 배열이다. 

PHP 는 둘, 셋, 넷, 다섯, 그리고 그 이상의 다차원 배열을 이해한다. 그러나 세 가지 단계 보다 더 깊은 배열은 대부분의 사람에게 관리가 어렵다.  

  배열의 차원은 요소를 선택하는데 필요한 인덱스의 수를 나타냅니다. 
  • 2 차원 배열의 경우, 요소를 선택하기 위해서는 두 개의 인덱스가 필요
  • 3 차원 배열의 경우, 요소를 선택하기 위해서는 세 개의 인덱스가 필요

PHP - 2-차원 Arrays

2 차원 배열은 "배열들의 배열"이다. (3 차원 배열은 "배열들의 배열들의 배열").

먼저, 다음 표를 살펴보자:

Name Stock Sold
Volvo 22 18
BMW 15 13
Saab 5 2
Land Rover 17 15

위의 표는 다음과 같이 2 차원 배열에 저장할 수 있다.:

$cars = array
  (
  array("Volvo",22,18),
  array("BMW",15,13),
  array("Saab",5,2),
  array("Land Rover",17,15)
  );

이제 2 차원 $cars 배열은 4개의 배열을 포함하며, 두 개의 인덱스(행과 열)을 갖는다.

$cars 배열의 요소에 접근하려면, 두 개의 인덱스(행과 열)를 지정해야만 한다.:

Example

<?php
echo $cars[0][0].": In stock: ".$cars[0][1].", sold: ".$cars[0][2].".<br>";
echo $cars[1][0].": In stock: ".$cars[1][1].", sold: ".$cars[1][2].".<br>";
echo $cars[2][0].": In stock: ".$cars[2][1].", sold: ".$cars[2][2].".<br>";
echo $cars[3][0].": In stock: ".$cars[3][1].", sold: ".$cars[3][2].".<br>";
?>

Run example »

우리는 또한 $cars 배열의 요소를 얻기 위해 For 루프 안에 또 다른 For 루프를 넣을 수 있다. (우리는 여전히 두 개의 인덱스를 가리키도록 해야 한다.):

Example

<?php
for ($row = 0; $row < 4; $row++)
  {
  echo "<p><b>Row number $row</b></p>";
  echo "<ul>";
  for ($col = 0; $col < 3; $col++)
    {
    echo "<li>".$cars[$row][$col]."</li>";
    }
  echo "</ul>";
  }
?>
