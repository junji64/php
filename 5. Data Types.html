PHP Strings

string 은 "Hello world!" 과 같은 일련의 문자들 입니다.

string 은 인용부호 안의 어떤 택스트도 될 수 있습니다. 당신은 단일 또는 이중 따옴표를 사용할 수 있습니다:

Example

<?php
$x = "Hello world!";
$y = 'Hello world!';

echo $x;
echo "<br>";
echo $y;
?>

PHP Integers

integer 는 소수점(decimals)이 없는 숫자입니다.

Rules for integers:

  • integer 는 적어도 하나의 숫자가 있어야 함(0-9)
  • integer 는 쉼표나 공백을 포함할 수 없다.
  • integer 는 소수점이 없어야 한다.
  • integer 는 양수 또는 음수가 될 수있다.
  • Integers 는 세가지 형식으로 지정 할 수 있다.: 10진수(decimal : 10-based), 16진수(hexadecimal :16-based - prefixed with 0x) or 8진수(octal : 8-based - prefixed with 0)

다음의 예에서 $x 는 정수입니다.   PHP var_dump() 함수는 변수의 데이터 형식과 값을 반환합니다.:

Example

<?php
$x = 5985;
var_dump($x);
?>

PHP Floating Point Numbers

부동 소수점 숫자(floating point number)는 소수점 또는 지수 형태의 숫자이다.

다음 예에서 $x 는 부동 소수점입니다. PHP var_dump() 함수는 변수의 데이터 형식과 값을 반환한다.:

Example

<?php
$x = 10.365;
var_dump($x);
?>

PHP Booleans

Booleans 은 TRUE 또는 FALSE 가 될 수 있다.

$x=true;
$y=false;

Booleans 은 종종 조건 시험에 사용된다. 

PHP Arrays

array 는 하나의 변수에 여러 값을 저장한다.

다음 예에서는 배열을 생성하고, 배열의 데이터 형식과 값을 반환하는데 PHP var_dump() 함수를 사용한다.:

Example

<?php
$cars=array("Volvo","BMW","Toyota");
var_dump($cars);
?>

PHP Objects

객체(object)는 데이터와 해당 데이터를 처리하는 방법에 대한 정보를 저장하는 데이터 형식이다.

PHP 에서 객체는 명시적으로 선언되어야 한다. 

처음에는, 객체의 클래스(class)를 선언해야 한다. 이를 위해,  class 키워드를 사용한다.  class 는 속성(properties)과 메서드(method)를 포함할 수 있는 구조이다.

그 다음, 객체 클래스의 데이타 유형을 정의하고, 그 때 그 클래스의 인스턴스(instances)의 데이타 형식을 사용한다.:

Example

<?php
class Car
{
  var $color;
  function Car($color="green")
  {
    $this->color = $color;
  }
  function what_color()
  {
    return $this->color;
  }
}
?>

Run example »

You will learn more about objects in a later chapter of this tutorial.

PHP NULL Value

특별한 NULL 값은 변수에 값이 없음을 나타낸다. NULL 은 데이터 형식 NULL 의 유일한 가능한 값이다.

NULL 값은 변수가 비어 있는 지 아닌 지를 식별한다. 또한 빈 문자열과 데이터베이스의 null 값을 구별하는데 유용하 다.

변수는 값을  NULL로 설정하여 비워질 수 있다.:

Example

<?php
$x="Hello world!";
$x=null;
var_dump($x);
?>
