문자열(string) 은 "Hello world!" 와 같은 일련의 문자들 입니다.

PHP String Functions

문자열을 조작하는데 일반적으로 사용되는 몇몇 함수를 살펴본다.

strlen() function

strlen() 함수는 문자열의 길이를 반환한다.

다음의 예는 문자열  "Hello world!"의 길이를 반환한다.:

Example

<?php
echo strlen("Hello world!");
?>

Run example »

Tip: strlen() 는 문자열이 언제 끝나는 지를 아는 것이 중요할 때 루프나 다른 함수에서 종종 사용된다. (i.e. 루프에서, 문자열의 마지막 문자 다음에 루프를 중지하기를 원할 수 있다).

str_word_count() function

str_word_count() 함수는 문자열의 단어 수를 계산합니다.

다음의 예는 문자열  "Hello world!"의 길이를 반환한다.:

Example

<?php
echo str_word_count("Hello world!"); // outputs 2
?>

Run example »

strrev() function

strrev() 함수는 문자열을 뒤집습니다.

다음의 예는 문자열 "Hello world!"를 뒤집습니다.

Example

<?php
echo strrev("Hello world!"); // outputs !dlrow olleH
?>

Run example »

PHP strpos() function

strpos() 함수는 문자열에서 지정된 문자나 텍스트를 검색하는 데 사용된다.

매치가 발견되면, 첫번째 매치의 문자 위치를 반환한다. 매치가 발견되지 않으면,  FALSE 를 반환한다.

아래의 예제는 "Hello world!" 문자열에서 텍스트 "world" 를 검색한다.:

Example

<?php
echo strpos("Hello world!","world");
?>

Run example »

위의 코드의 출력은 다음과 같다: 6.

Tip: 위의 예에서 문자열 "world" 의 위치는 6 입니다. 그것이 (7이 아닌) 6인 이유는 문자열의 첫 번째 문자의 위치는 (1이 아닌) 0 이기 때문입니다.    

str_replace() function

str_replace() 함수는 문자열에서 일부 문자를 다른 문자로 대체합니다.

아래의 예제는 텍스트 "world"를 "Dolly"로 바꿉니다.:

Example

<?php
echo str_replace("world", "Dolly", "Hello world!"); // outputs Hello Dolly!
?>

Run example »

substr() function

substr() 함수는 문자열의 일부를 반환합니다.

아래의 예제는 문자열에서 "world"를 반환합니다.

Example

<?php
echo substr("Hello world",6);
?>

Run example »

strtolower() function

strtolower() 함수는 문자열을 소문자로 변환합니다.

아래의 예제는 모든 문자를 소문자로 변환 :

Example

<?php
echo strtolower("Hello WORLD.");
?>

Run example »

strtoupper() function

strtoupper() 함수는 문자열을 대문자로 변환합니다.

아래의 예제는 모든 문자를 대문자로 변환 :

Example

<?php
echo strtoupper("Hello WORLD!");
?>

Run example »

관련 함수 :

  • lcfirst() - 문자열의 첫 번째 문자를 소문자로 변환
  • ucfirst() -문자열의 첫 문자를 대문자로 변환
  • ucwords() -문자열에있는 각 단어의 첫 문자를 대문자로 변환합니다.

trim() function

trim() 함수는 문자열의 양쪽에서 공백과 기타 미리 정의 된 문자를 제거합니다.

아래의 예제는 문자열의 양쪽에서 문자를 제거합니다 ( "Hello"의 "He"및 "World"의 "d!"):

Example

<?php
$str = "Hello World!";
echo $str . "<br>";
echo trim($str,"Hed!");
?>

Run example »

explode() function

explode() 함수는 문자열을 배열로 나눕니다.

아래의 예제는 문자열을 배열로 나눕니다.:

Example

<?php
$str = "Hello world. It's a beautiful day.";
print_r (explode(" ",$str));
?>

Run example »

implode() function

implode() 함수는 배열 요소에서 문자열을 반환합니다.

아래의 예제는 문자열로 배열 요소 결합 :

Example

<?php
$arr = array('Hello','World!','Beautiful','Day!');
echo implode(" ",$arr);
?>

Run example »

strcmp() function

strcmp() 함수는 두 문자열을 비교합니다.

아래의 예제는 두 문자열 비교 (대소 문자 구분) :

Example

<?php
echo strcmp("Hello world!","Hello world!");
?>
