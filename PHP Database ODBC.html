ODBC 는 데이타 소스(e.g. an MS Access database)에 연결할 수 있도록 응용 프로그램 인터페이스(Application Programming Interface : API) 이다.

Create an ODBC Connection

With an ODBC 연결을 사용하면, ODBC 연결이 허용되는 한, 네트워크의 모든 컴퓨터에서 데이타베이스에 연결할 수 있다. 

ODBC MS Access 데이타베이스에 ODBC 연결을 생성하는 방법 : 

  1. 제어판(Control Panel)에서 관리도구(Administrative Tools)를 연다.
  2. Double-click on the Data Sources (ODBC) icon inside.
  3. Choose the System DSN tab.
  4. Click on Add in the System DSN tab.
  5. Select the Microsoft Access Driver. Click Finish.
  6. In the next screen, click Select to locate the database.
  7. Give the database a Data Source Name (DSN).
  8. Click OK.

이 구성은 당신의 웹사이트가 있는 컴퓨터에서 수행해야 합니다. 당신 자신의 컴퓨터에 Internet Information Server (IIS)를 실행하는 겅우에는, 위의 설명이 작동하지만, 당신의 웹사이트가 원격 서버에 있는 경우에는 해당 서버에 물리적으로 접근하거나, 웹 호스트에게 당신이 사용할 수 있도록  DSN 설정을 요청해야 합니다. 

Connecting to an ODBC

odbc_connect() 함수가 ODBC 데이타 소스에 연결하는 데 사용됩니다.  함수는 네 개의 매개변수를 취합니다: the data source name, username, password, and an optional cursor type.

odbc_exec() 함수는 SQL 문을 실행하는 데 사용됩니다.

Example

다음 예제에서는 northwind 라 부르는 DSN에 사용자 이름과 암호 없이 연결을 생성합니다. 그리고 SQL 을 생성하여 실행합니다.:

$conn=odbc_connect('northwind','','');
$sql="SELECT * FROM customers";
$rs=odbc_exec($conn,$sql);

Retrieving Records

odbc_fetch_row() 함수는 result-set 으로부터 레코드를 반환하는 데 사용됩니다. 이 함수는 행들을 반환할 수 있으면 true 를, 그렇지 않으면 false 를 반환한다. 

이 함수는 두 개의 파라메타를 사용한다 : ODBC 결과 식별자와 옵셔널 행 번호를 :

odbc_fetch_row($rs)

Retrieving Fields from a Record

odbc_result() 함수는 레코드로부터 필드를 읽는 데 사용된다. 이 함수는 두 개의 매개변수를 가진다.: ODBC 결과 식별자(result identifier) 와 필드 번호 또는 이름.

아래의 코드 라인은 레코드로부터 첫 번째 필드의 값을 반환한다. 

$compname=odbc_result($rs,1);

아래의 코드 라인은 "CompanyName"이라고 부르는 필드의 값을 반환한다. :

$compname=odbc_result($rs,"CompanyName");

Closing an ODBC Connection

odbc_close() 함수는 ODBC 연결을 종료하는데 사용된다.

odbc_close($conn);

An ODBC Example

다음의 예는 데이타베이스에 연결하고, 결과 집합을 생성한 다음, HTML  테이블의 데이타로 표시하는 방법을 보여준다. 

<html>
<body>

<?php
$conn=odbc_connect('northwind','','');
if (!$conn)
  {exit("Connection Failed: " . $conn);}
$sql="SELECT * FROM customers";
$rs=odbc_exec($conn,$sql);
if (!$rs)
  {exit("Error in SQL");}
echo "<table><tr>";
echo "<th>Companyname</th>";
echo "<th>Contactname</th></tr>";
while (odbc_fetch_row($rs))
  {
  $compname=odbc_result($rs,"CompanyName");
  $conname=odbc_result($rs,"ContactName");
  echo "<tr><td>$compname</td>";
  echo "<td>$conname</td></tr>";
  }
odbc_close($conn);
echo "</table>";
?>

</body>
</html>
