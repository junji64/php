PHP while 루프(loops)는 지정 된 조건이 true 인 동안 코드 블록을 실행한다. 

PHP for 루프(loops)는 지정된 횟수만큼 코드 블록을 실행한다.

PHP Loops

코드를 작성할 때 종종 동일한 코드 블록을 연속해서 다시 실행하기를 원하는 경우가 있다. 스크립트에 거의 동일한 코드를 여러번 추가 하는 대신에, 이러한 작업을 수행하기 위해서 루프를 사용할 수 있다. 

PHP 에는 다음과 같은 루프문장들이 지원된다.:

  • while - 지정된 조건이 true 인 동안 코드블록을 반복
  • do...while - 코드블록을 한번 실행한 후에, 지정된 조건이 true 인 동안 루프를 반복
  • for - 지정된 횟수 만큼 코드블록을 반복
  • foreach -배열의 각 요소에 대하여 코드블록을 반복

The PHP while Loop

while 루프는 지정된 조건이 true 인 동안 코드블록을 실행합니다.

Syntax

while (condition is true)
  {
  code to be executed;
  }

아래 예제에서 첫 번째는 변수 $x 를 1 로 ($x=1;) 놓고. $x 가 5 보다 작거나 같은 동안 루프를 실행한다. $x 는 루프가 실행될 때 마다 1씩 증가($x++;) 한다. :

Example

<?php
$x=1;
while($x<=5)
  {
  echo "The number is: $x <br>";
  $x++;
  }
?>

The PHP do...while Loop

do...while 루프는 항상 코드블록을 한번 실행하고 나서 조건을 확인합니다. 그리고 지정된 조건이 true 인 동안 루프를 반복합니다. 

Syntax

do
  {
  code to be executed;
  }
while (condition is true);

아래 예제에서 첫 번째는 변수 $x 를 1 로 ($x=1;) 놓습니다. 그리고 몇가지 출력을 하고 변수 $x 를 1 증가 시킵니다. 그 다음 조건을 확인합니다 ($x 가 5 보다 작거나 같은 지를), 그리고 루프는 $x 가 5 보다 작거나 같은 동안 루프를 실행을 계속 합니다. :

Example

<?php
$x=1;
do
  {
  echo "The number is: $x <br>";
  $x++;
  }
while ($x<=5)
?>

Run example »

do while 루프에서는 루프 내의 문장이 실행되 후에 조건을 시험하는 것을 주목하시오.  이는 조건이 처음부터 실패하더라도 do while 루프에서는적어도 한번은 그 문장이 실행되는 것을 의미 합니다. 

아래의 예제는 The example below sets the $x 변수를 6 으로 설정합니다. 그리고 루프가 실행되고 그 다음에 조건이 확입됩니다. :

Example

<?php
$x=6;
do
  {
  echo "The number is: $x <br>";
  $x++;
  }
while ($x<=5)
?>

The PHP for Loop

for 루프는 스크립트가 실행되야 할 횟수를 미리 알고 있는 경우에 사용됩니다.

Syntax

for (init counter; test counter; increment counter)
  {
  code to be executed;
  }

Parameters:

  • init counter: 루프 카운터 값을 초기화
  • test counter: 각 루프 반복을 평가. 이것이 TRUE 로 평가되면 루프가 계속됩니다. 이것이 FALSE 로 평가되면 루프가 끝납니다.
  • increment counter: 루프 카운터 값을 증가

아래의 예는 0 부터 10 까지의 숫자를 표시합니다.:

Example

<?php
for ($x=0; $x<=10; $x++)
  {
  echo "The number is: $x <br>";
  }
?>

The PHP foreach Loop

foreach 루프는 배열에서만 작동되며, 배열의 각 키/값 쌍(key/value pair)에 대하여 반복하는데 사용된다. 

Syntax

foreach ($array as $value)
  {
  code to be executed;
  }

각 루프 반복에 대하여 현재 배열 요소의 값은 $value 에 할당되고, 배열 포인터는 마지막 배영 요소에 이를 때 까지 하나씩 이동한다.

다음의 예는 주어진 배열($colors)의 값을 출력하는 루프를 보여준다. :

Example

<?php
$colors = array("red","green","blue","yellow");
foreach ($colors as $value)
  {
  echo "$value <br>";
  }
?>
