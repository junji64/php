MySQL 데이터베이스에서 데이터 선택 및 필터링

WHERE 절은 레코드를 필터링하는 데, 또는 지정된 기준을 충족하는 레코드 만을 추출하는데 사용된다.

SELECT column_name(s)
FROM table_name
WHERE column_name operator value

위의 문을 PHP 가 실행하게 하기 위해서는 mysqli_query() 함수를 사용해야만 한다. 이 함수는 MySQL 연결에 쿼리나 명령을 보낼 때 사용된다.

Operators in The WHERE Clause

다음의 연산자들을  WHERE  문에서 사용할 수 있다.:

Operator Description
= Equal
<> Not equal. Note: In some versions of SQL this operator may be written as !=
> Greater than
< Less than
>= Greater than or equal
<= Less than or equal
BETWEEN Between an inclusive range
LIKE Search for a pattern
IN To specify multiple possible values for a column

MySQLi로 데이터 선택 및 필터링

다음 예제는 성이 "Doe"인 MyGuests 테이블에서 id, firstname 및 lastname 열을 선택하여 페이지에 표시합니다.:

Example (MySQLi Procedural)

<?php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "myDB";

// Create connection
$conn = mysqli_connect($servername, $username, $password, $dbname);
// Check connection
if (!$conn) {
die("Connection failed: " . mysqli_connect_error());
}

$sql = "SELECT id, firstname, lastname FROM MyGuests WHERE lastname='Doe'";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
// output data of each row
while($row = mysqli_fetch_assoc($result)) {
echo "id: " . $row["id"]. " - Name: " . $row["firstname"]. " " . $row["lastname"]. "<br>";
}
} else {
echo "0 results";
}

mysqli_close($conn);
?>
