이 장에서는 서버에 파일을 생성하고 쓰는 방법을 설명한다. 

PHP Create File - fopen()

fopen() 함수가 파일을 생성하는데 사용됩니다. 약간은 혼란 스러울 수 있지만, PHP 에서는 파일을 열 때 사용하는 함수와 동일한 함수를 사용하여 파일을 생성합니다.

존재 하지 않는 파일에 fopen() 를 사용하면, 그 파일을 생성하여 쓰거나(writing (w)) 추가(appending (a))가 가능하도록 해줍니다. 

아래의 예제는 "testfile.txt"라고 하는 새로운 파일을 생성합니다. 이 파일은 PHP 코드가 있는 곳과 같은 디렉토리에 생성됩니다.:

Example

$myfile = fopen("testfile.txt", "w")


PHP File 권한(Permissions)

이 코드를 실행하려 할 때 에러가 발생하면, 하드 드라이브에 정보를 기록하는 권한을 PHP  파일에 부여 했는 지를 확인합니다. 

PHP Write to File - fwrite()

fwrite() 함수는 파일에 기록하는데 사용됩니다.

fwrite() 의 첫번째 파라메타는 쓸 파일 이름을, 두 번째 파라메타는 쓸 문자열 입니다. 

아래의 예제는 "newfile.txt" 라고 부르는 새로운 파일에 이름 몇 개를 씁니다.  :

Example

<?php
$myfile = fopen("newfile.txt", "w") or die("Unable to open file!");
$txt = "John Doe\n";
fwrite($myfile, $txt);
$txt = "Jane Doe\n";
fwrite($myfile, $txt);
fclose($myfile);
?>

파일 "newfile.txt"에 두 번 쓴 것을 주목하시오. 파일에 쓸 때, 첫번째는 "John Doe" 를, 두번째는 "Jane Doe"를 갖는 문자열 $txt 를 보냅니다. 작성 완료 후,  fclose() 함수를 사용해서 파일을 닫습니다. 

 "newfile.txt" 파일을 열어보면, 다음과 같을 것이다.:

John Doe
Jane Doe

PHP Overwriting

이제 "newfile.txt" 는 데이터를 포합하고 있으므로, 우리가 쓰기 위해서 기존의 파일을 열 때 어떤 일이 일어나는지 보일 수 있습니다. 모든 기존의 데이터가 삭제되며, 빈 파일로 시작하게 된다. 

아래의 예제에서 우리는 기존의 파일 "newfile.txt"을 열고, 그 곳에 어떤 새로운 데이타를 씁니다. :

Example

<?php
$myfile = fopen("newfile.txt", "w") or die("Unable to open file!");
$txt = "Mickey Mouse\n";
fwrite($myfile, $txt);
$txt = "Minnie Mouse\n";
fwrite($myfile, $txt);
fclose($myfile);
?>

 "newfile.txt" 파일을 이제 다시 열면, John 과 Jane 은 없어지고, 단지 우리가 쓴 새로운 데이터 만이 존재한다. :

Mickey Mouse
Minnie Mouse
