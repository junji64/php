AJAX 는 웹 페이지 전체를 다시 로드하지 않고, 일부를 업데이트하는 방법에 대한 것이다.

What is AJAX?

AJAX = Asynchronous JavaScript and XML (비동기 자바스크립트와 XML).

AJAX는 빠르고 동적인 웹 페이지를 만들기 위한 기술이다.

AJAX는 보이지 않게 뒤에서 서버와 적은 양의 데이터를 교환하여 비동기적으로 웹 페이지가 업데이트 되게 합니다. 이는 전체 페이지를 다시 로드하지 않고, 웹 페이지의 일부를 업데이트하는 것이 가능하다는 것을 의미한다.

클래식 웹 페이지(AJAX를 사용하지 않는)는 내용을 변경해야하는 경우 전체 페이지를 다시 로드해야합니다.

AJAX를 사용하는 응용 프로그램의 예 : Google Maps, Gmail, Youtube, 과 Facebook tabs.

How AJAX Works

AJAX is Based on Internet Standards

AJAX는 인터넷 표준에 근거하고, 다음의 것들을 조합하여 사용한다 :

  • XMLHttpRequest object (서버와 비동기적으로 데이터를 교환하기 위해)
  • JavaScript/DOM (정보를 표시/상호작용 하기위해)
  • CSS (데이터를 스타일하기 위해)
  • XML (데이터를 전송하기 위한 포맷으로 종종 사용)
 AJAX 응용 프로그램은 브라우저 및 플랫폼에 독립적이다!

Google Suggest

AJAX는 구글 제안으로, 구글에 의해 2005 년에 널리 사용되게 되었습니다.

구글 제안(Google Suggest)은 매우 동적인 웹 인터페이스를 생성하기위해 AJAX를 사용하자는 것이다 : 당신이 구글의 검색 창에 입력을 시작하면, 자바 스크립트는 서버에 문자를 보내고 서버는 제안 목록을 반환 하는 것이다.
