PHP 에서 출력을 하는 두 가지 기본적인 방법:  echo 와 print.

PHP echo 와 print Statements

echo 와 print 사이에 약간의 차이가 있습니다.:

  • echo - 하나 이상의 문자열을 출력할 수 있다.
  • print - 오직 하나의 문자열 만을 출력할 수 있으며, 항상 1 을 반환한다.
Tip: echo 는 어떤 값도 반환하지 않으므로, print 에 비하여 약간 빠르다.

PHP echo Statement

echo 는 언어 구성소(language construct)이며, 괄호와 함께 또는 없이 사용될 수 있다.: echo or echo().

Display Strings

다음의 예에서 echo 명령으로 여러 다른 문자열을 표시하는 것을 (또한 문자열이 HTML 태그를 포함할 수 있음도) 보여 줍니다.  :

Example

<?php
echo "<h2>PHP is fun!</h2>";
echo "Hello world!<br>";
echo "I'm about to learn PHP!<br>";
echo "This", " string", " was", " made", " with multiple parameters.";
?>

Display Variables

다음의 예에서는 echo 명령으로 문자열과 변수를 표시하는 방법을 보여줍니다. :

Example

<?php
$txt1 = "Learn PHP";
$txt2 = "W3Schools.com";
$x = 5;
$y = 4;

echo "

" . $txt1 . "

";
echo "Study PHP at " . $txt2 . "
";
echo $x + $y;
?>

PHP print Statement

print 또한 언어 구성소(language construct)이며, 괄호와 함께 또는 없이 사용될 수 있다: print or print().

Display Strings

다음의 예에서 print 명령으로 여러 다른 문자열을 표시하는 것을 (또한 문자열이 HTML 태그를 포함할 수 있음도) 보여 줍니다.:

Example

<?php
print "<h2>PHP is fun!</h2>";
print "Hello world!<br>";
print "I'm about to learn PHP!";
?>

Run example »

Display Variables

다음의 예에서는 print 명령으로 문자열과 변수를 표시하는 방법을 보여줍니다.
이중 따옴표 " " 안의 string들은 해석된다. 즉, 그 안의 변수들은 그들의 값으로 string에 포함된다.
애매함을 피하기 위해서, 중괄호 {} 로 변수를 감쌀 수 있다. :

Example

<?php
$txt1="Learn PHP";
$txt2="W3Schools.com";
$cars=array("Volvo","BMW","Toyota");

print $txt1;
print "<br>";
print "Study PHP at $txt2";
print "My car is a {$cars[0]}";
?>

Run example »

단일 따옴표' ' 안의 string들은 해석되지 않는다. 즉, 그 안의 변수들은 그 들의 값으로 string에 포함된다.

<?php
$age = 16;
print 'You are $age years old.\n';    // You are $AGE years old.\N
?>

Embedded PHP

print 문으로 HTML 태그를 프린트하는 것은 나쁜 스타일 이며 에러 발생이 쉽다.

<?php
print "<!DOCTYPE html>\n";
print "<html>\n";
print "  <head>\n";
print "    <title>Geneva's web page</title>\n";
...
for ($i = 1; $i <= 10; $i++) {
  print "<p class=\"count\"> I can count to $i! </p>\n";
}
?>

PHP expresion blocks

<?= expr ?> (PHP expression block)은 식을 평가하여 그 값을 HTML에 포함시킨다.
<?= expr ?>  은  <?php print expr; ?> 과 같다.:

<!DOCTYPE html>
<html>
  <head><title>CSE 190 M: Embedded PHP</title></head>  
  <body>
    <?php for ($i = 99; $i >= 1; $i--) { ?>
      <p> <?= $i ?> bottles of beer on the wall, <br />
          <?= $i ?> bottles of beer. <br />
          Take one down, pass it around, <br />
          <?= $i - 1 ?> bottles of beer on the wall. </p>
    <?php } ?>
  </body>
</html>

expression block 은 HTML 태그와 속성 안에 올 수 있다. :

<!DOCTYPE html>
<html>
  <head><title>CSE 190 M: Embedded PHP</title></head>  
  <body>
    <?php for ($i = 1; $i <= 5; $i++) { ?>
      <h<?= $i ?>>This is a level <?= $i ?> heading.</h<?= $i ?>>
    <?php } ?>
  </body>
</html>
