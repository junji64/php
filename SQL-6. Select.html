SELECT 문은 데이타베이스로부터 데이타를 선택하는데 사용됩니다.

Select Data From a Database Table

SELECT 문은하나 이상의 테이블에서 데이터를 선택하는 데 사용됩니다.

Syntax

SELECT column_name(s) FROM table_name

또는 * 문자를 사용하여 테이블에서 모든 열을 선택할 수 있습니다.

SELECT * FROM table_name

MySQLi로 데이터 선택

다음 예제는 MyGuests 테이블에서 id, firstname 및 lastname 열을 선택하여 페이지에 표시합니다.

Example (MySQLi Procedural)

<?php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "myDB";

// Create connection
$conn = mysqli_connect($servername, $username, $password, $dbname);
// Check connection
if (!$conn) {
die("Connection failed: " . mysqli_connect_error());
}

$sql = "SELECT id, firstname, lastname FROM MyGuests";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
// output data of each row
while($row = mysqli_fetch_assoc($result)) {
echo "id: " . $row["id"]. " - Name: " . $row["firstname"]. " " . $row["lastname"]. "<br>";
}
} else {
echo "0 results";
}

mysqli_close($conn);
?>

먼저 MyGuests 테이블에서 id, firstname 및 lastname 열을 선택하는 SQL 쿼리를 설정합니다. 다음 코드 줄은 쿼리를 실행하고 결과 데이터를 $result라는 변수에 넣습니다.

그런 다음 function num_rows()의 반환 된 행이 0 개 이상인지 확인합니다.

반환 된 행이 0 개 이상인 경우에는 함수 fetch_assoc()은 모든 결과를 반복 할 수 있는 연관 배열에 넣습니다. while()루프는 결과 세트에 대하여 반복하여 ID, 이름 및 성 열로부터 데이터를 출력한다.

PHP 가 위의 문을 실행하게 하려면 mysqli_query() 함수를 사용하여야 한다. 이 함수는 MySQL 연결에 쿼리나 명령을 보낼 때 사용된다. 

Example

다음의 예는 "Persons" table에 저장된 모든 데이타를 선택한다. (The * character selects all the data in the table):

<?php
$con=mysqli_connect("example.com","peter","abc123","my_db");
// Check connection
if (mysqli_connect_errno())
  {
  echo "Failed to connect to MySQL: " . mysqli_connect_error();
  }

$result = mysqli_query($con,"SELECT * FROM Persons");

while($row = mysqli_fetch_array($result))
  {
  echo $row['FirstName'] . " " . $row['LastName'];
  echo "<br>";
  }

mysqli_close($con);
?>

위의 예는 mysqli_query() 함수에 의해서 $result 변수로 반환된 데이타를 저장한다.

다음에는, recordset 으로부터 첫번째 행을 배열로 반환하는데 mysqli_fetch_array() 함수를 사용한다. mysqli_fetch_array() 를 호출 할 때마다 recordset 에서 다음 행을 반환한다. while 루프는 recordset 에서의 모든 레코드를 반복한다. 각 행의 값을 프린트하기 위해서, 우리는 PHP $row 배열 변수 ($row['FirstName'] and $row['LastName'])를 사용한다. 

위의 코드의 출력을 다음과 같을 것이다.:

Peter Griffin
Glenn Quagmire

Display the Result in an HTML Table

다음 예는 위의 예에서와 같은 데이타를 선택하지만, 데이타를 HTML table 형식으로 표시한다.:

<?php
$con=mysqli_connect("example.com","peter","abc123","my_db");
// Check connection
if (mysqli_connect_errno())
  {
  echo "Failed to connect to MySQL: " . mysqli_connect_error();
  }

$result = mysqli_query($con,"SELECT * FROM Persons");

echo "<table border='1'>
<tr>
<th>Firstname</th>
<th>Lastname</th>
</tr>";

while($row = mysqli_fetch_array($result))
  {
  echo "<tr>";
  echo "<td>" . $row['FirstName'] . "</td>";
  echo "<td>" . $row['LastName'] . "</td>";
  echo "</tr>";
  }
echo "</table>";

mysqli_close($con);
?>

위의 코드의 출력을 다음과 같을 것이다:

Firstname Lastname
Glenn Quagmire
Peter Griffin
