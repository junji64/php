MySQL 데이터베이스에서 데이터 선택 및 정렬

ORDER BY keyword 는 record set 에서 데이타를 정렬하는데 사용된다.

ORDER BY keyword 는 기본적으로 레코드를 오름차순으로 정렬한다. 만약 레코드를 내림차순으로 정렬하고 싶다면, DESC keyword 를 사용할 수 있다.

Syntax

SELECT column_name(s)
FROM table_name
ORDER BY column_name(s) ASC|DESC

SQL 에 대한 자세한 내용은 ==> SQL tutorial.

MySQLi로 데이터 선택 및 정렬

다음 예에서는 MyGuests 테이블에서 id, firstname 및 lastname 열을 선택합니다. 레코드는 lastname 열을 기준으로 정렬됩니다.:

Example (MySQLi Procedural)

<?php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "myDB";

// Create connection
$conn = mysqli_connect($servername, $username, $password, $dbname);
// Check connection
if (!$conn) {
die("Connection failed: " . mysqli_connect_error());
}

$sql = "SELECT id, firstname, lastname FROM MyGuests ORDER BY lastname";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
// output data of each row
while($row = mysqli_fetch_assoc($result)) {
echo "id: " . $row["id"]. " - Name: " . $row["firstname"]. " " . $row["lastname"]. "<br>";
}
} else {
echo "0 results";
}

mysqli_close($conn);
?>

Run example »

위의 코드의 출력을 다음과 같을 것이다:

id: 1 - Name: John Doe
id: 3 - Name: Julie Dooley
id: 2 - Name: Mary Moe

Order by Two Columns

하나 이상의 열로 순서짖는 것도 가능하다. 하나 이상의 열로 순서를 지을 때는 두번째 열은 첫번째 열의 결과가 같을 경우에만 사용된다. :

SELECT column_name(s)
FROM table_name
ORDER BY column1, column2
