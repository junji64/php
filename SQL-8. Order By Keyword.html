The ORDER BY Keyword

ORDER BY keyword 는 record set 에서 데이타를 정렬하는데 사용된다.

ORDER BY keyword 는 기본적으로 레코드를 오름차순으로 정렬한다.

만약 레코드를 내림차순으로 정렬하고 싶다면, DESC keyword 를 사용할 수 있다.

Syntax

SELECT column_name(s)
FROM table_name
ORDER BY column_name(s) ASC|DESC

SQL 에 대한 자세한 내용은 ==> SQL tutorial.

Example

다음의 예는 "Persons" 테이블에 저장된 모든 데이타를 선택하고 ,  "Age" 열의 결과로 정렬한다.:

<?php
$con=mysqli_connect("example.com","peter","abc123","my_db");
// Check connection
if (mysqli_connect_errno())
  {
  echo "Failed to connect to MySQL: " . mysqli_connect_error();
  }

$result = mysqli_query($con,"SELECT * FROM Persons ORDER BY age");

while($row = mysqli_fetch_array($result))
  {
  echo $row['FirstName'];
  echo " " . $row['LastName'];
  echo " " . $row['Age'];
  echo "<br>";
  }

mysqli_close($con);
?>

위의 코드의 출력을 다음과 같을 것이다:

Glenn Quagmire 33
Peter Griffin 35

Order by Two Columns

하나 이상의 열로 순서짖는 것도 가능하다. 하나 이상의 열로 순서를 지을 때는 두번째 열은 첫번째 열의 결과가 같을 경우에만 사용된다. :

SELECT column_name(s)
FROM table_name
ORDER BY column1, column2
